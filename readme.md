## Recursion Sum

### Example

```javascript
const arr = [1, 2, 3, 4, 5, 6, 7, 8, 9];
/**
 * Sum function using recursion
 * @param arr number[] array of numbers to get the sum of
 * @returns number The sum of the array numbers
 */
Lib.Math.recursionSum(arr);
```
